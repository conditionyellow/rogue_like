How to Play

Commands:

help: Show available commands

language [en|ja|1|2]: Change language

Movement:

move [n|s|e|w|ne|nw|se|sw]: Move player

rest: Wait/Rest (HP/MP recovery)

Items:

get: Pick up item

inventory: Open Inventory UI

use [index|name]: Use item

equip [index|name]: Equip item

unequip [slot]: Unequip item (weapon, armor, shield)

Magic:

magic: Open Magic selection

Shop:

shop: Interact with Merchant

Game:

save: Save Game

load: Load Game
